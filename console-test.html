Test Console localStorage

Ouvrez la console développeur (F12) et testez les commandes ci-dessous :

Commandes à tester dans la console :

// 1. Ajouter un plan au localStorage
const plan = {
  "id": "68bf677bdf8ea2b424543a03",
  "name": "Plan Enterprise",
  "description": "Plan complet pour les grandes organisations",
  "billingCycle": "MONTHLY",
  "price": 99.99,
  "currency": "EUR",
  "includedFeatures": ["Fonctionnalité Premium", "Support 24/7"]
};
localStorage.setItem('selectedPlan', JSON.stringify(plan));

// 2. Vérifier que le plan est bien stocké
console.log('Plan stocké:', JSON.parse(localStorage.getItem('selectedPlan')));

// 3. Aller à la page de configuration
